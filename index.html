InStock

A collaboration projects and team effort, InStock was built under a strict timeline by 5 developpers using JIRA boards for management and GIT (Bitbucket) for version control. InStock features a Front-end built in REACT.JS and a RESTful API built with EXPRESS.JS
Pixel-Perfectly rendered from very precise mockups, InStock was brought to life using React.JS and SCSS. Challenges included app responsiveness, modal functionality and mapping the data accurately throughout the app. Managing git branches as well as proper use of BEM principals were crucial to minimizing CSS or other conflicts and were large contributors to the on-time completion of the project.
Instock data is served by a custom RESTful API created by the team and React State/Props for distribution throughout the app. Challenges included architecture and routing. InStock supports multiple warehouses and inventory manipulation for each of them. InStock is fully scalable and deployable.
